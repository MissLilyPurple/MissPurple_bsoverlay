# Miss_Lily_Purple Beat Saber OBS Overlay

A lightweight, customizable Beat Saber song overlay for OBS, powered by **Beat Saber DataPuller**.

This overlay displays current song information such as:
- Song title and artist
- Album cover
- Difficulty
- Rank
- Combo
- Elapsed / total time

The overlay is designed to be used as a **Browser Source** in OBS and works locally with each user's own Beat Saber DataPuller instance.
